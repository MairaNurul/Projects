Proyek ini merupakan hasil pengerjaan ulang secara mandiri dari proyek kelompok yang pernah dikerjakan sebelumnya. Seluruh kode, analisis dalam repository ini telah disusun ulang, serta disempurnakan secara pribadi untuk memastikan orisinalitas dan peningkatan kualitas hasil akhir.

Project-Project ini merupakan Study Case pada Mata Kuliah Analisis Data Eksplorasi dan Pengantar Penambangan Data

1. Project Exploratory Data Analysis (EDA) –  Global Superstore

   Tujuan dari analisis ini untuk melakukan analisis pelanggan, produk dan tren penjualan untuk mengidentifikasi pola pembelian serta mengguanakan Python (pandas, matplotlib,      seaborn) untuk membersihkan data, membuat visualisasi, dan menemukan insight terkait perilaku konsumen. 

2. Project Exploratory Data Analysis & Strategic Recommendation — RPPI (Studi Kasus Internal)

   Tujuan dari analisis ini untuk melakukan EDA pada dataset listing properti (pembersihan data, pemeriksaan kualitas, visualisasi tren) serta analisis perbandingan harga
   antar-kota: agregasi median & mean price per m², distribusi harga, dan uji perbedaan untuk mengidentifikasi kota yang relatif mahal/murah.
